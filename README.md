# カラダ振る～お前もギタリストにしてやろうか～
"カラダ振る～お前もギタリストにしてやろうか～" developed by Ngai Man Chun

本格エアギターリズムゲーム
※著作権保護のため、楽曲（フリー音源除く）はミュート処理しています

## デモ動画
https://youtu.be/VMUpB2KFo5g

## ゲームの流れ
タイトル画面から[Enter]を押すと、曲選択画面に入ります。
曲選択画面で好きな曲を選んでください。
曲の選択以外、音の遅延補正と譜面速度も調節できます。
曲が決まったら[Enter]でゲーム開始です。

## 操作方法
[F1～F6]：ピッキングする音を決める

[Enter]：ピッキングする

[Esc長押し]：リタイア

流れてくるノーツに合わせて、
[F1～F6]を押さえながら[Enter]でピッキングをすればスコアが獲得ができます。

タイミングによってPERFECT、GREAT、COOL、GOOD、BADのクオリティに分けられています。

より正確なリズムでノーツを消せばより高い得点になります。

コンボを刻めばさらに得点がスケールアップさせます。

ノーツが▼▲マークが付いてる部分がピッキングするところです。

その後ろに付いてるヒレはサスティン（延長）といい、
ピッキングしたあとも[F1～F6]を押し続けるとサスティン部分を消せます。

基本的にコンボ数は▼▲マークが付いてる部分しかカウントされないので、
サスティンはボーナスとして考えた方が上手くプレイできるでしょう。
